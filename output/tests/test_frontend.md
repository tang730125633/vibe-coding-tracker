# 前端功能测试清单

## 测试环境准备

1. 启动后端服务: `cd output/backend && python main.py`
2. 启动前端服务: `cd output/frontend && npm run dev`
3. 访问: http://localhost:5173

## 1. 登录流程测试

### 1.1 登录成功
- [ ] 输入正确的用户名和密码（tang / tang123）
- [ ] 点击登录按钮
- [ ] 验证是否跳转到对应的首页（老师端或学员端）
- [ ] 验证 token 是否保存到 localStorage

### 1.2 登录失败
- [ ] 输入错误的密码
- [ ] 验证是否显示错误提示
- [ ] 输入不存在的用户名
- [ ] 验证是否显示错误提示

### 1.3 表单验证
- [ ] 用户名为空时，验证是否有提示
- [ ] 密码为空时，验证是否有提示
